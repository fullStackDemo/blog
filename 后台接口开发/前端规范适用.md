### 关于前端开发工具的格式代码问题

推荐适用 VSCODE 开发工具；

需要安装插件：editorConfig，配置如下：

```javascript
root = true

[*]
indent_style = space
indent_size = 4
charset = utf-8
trim_trailing_whitespace = false
insert_final_newline = false
```
